A backend engineer who ships

I'm a Backend Engineer at AlignBits LLC, working on an iPaaS platform where I build the services that move data reliably between enterprise systems. My stack is Java 17, Spring Boot, microservices, REST APIs, RabbitMQ, AWS SQS, PostgreSQL, Redis, and Docker.
